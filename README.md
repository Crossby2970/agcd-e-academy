# AGCD@E-Academy — Landing Page

Responsive landing page for AGCD@E-Academy, built with React + Vite + Tailwind CSS v4.

## Run locally

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build to dist/
npm run preview  # preview the production build
```

## Structure

- `src/data/` — editable content (pillars, courses, pathways, audiences, FAQs). Update copy here without touching components.
- `src/components/` — one component per section/UI element (see project deliverables notes for the full list).
- `src/index.css` — Tailwind import, AGCD brand theme (`@theme`), and base styles.

## Before going live

See the "Placeholder content" list in the delivery notes — founder photo, social links, `public/og-image.png`, impact-metric figures and testimonials all need real content before launch.
